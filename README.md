# Eletric-Calculator
O código pode ser executado sem utilização do terminal ou console
1. Calculadora de gastos por eletrodoméstico

Calcula o consumo de energia de um equipamento elétrico e seu custo mensal aproximado.

Como usar:

aperte o botão de calculo

Insira o nome do equipamento.

Informe a potência do equipamento e a unidade (W ou kW).

Caso insira W, o sistema converterá automaticamente para kW.

Se a unidade for inválida, o processo será reiniciado.

Insira quantas horas por dia o equipamento é usado.

Insira o valor cobrado por kWh (em reais).

O sistema exibirá um alerta com:

Consumo mensal aproximado em kWh.

Custo mensal aproximado em reais.

Exemplo:

Nome do item: Geladeira
Potência: 200
Unidade: W
Horas por dia: 24
Preço por kWh: 0.70

Resultado exibido:
"Seu/sua Geladeira usa 144.00 kWh, lhe custando um total de R$100.80"

2. conversor de unidades de eletroeletrônica

Converte valores entre unidades com prefixos métricos (G, M, k, m, μ(use u por questões técnicas), n).

Como usar:

Aperte o botão de conversão

Insira a unidade base completa (por exemplo, "W" ou "V").

Insira o valor a ser convertido.

Insira a unidade atual do valor, incluindo prefixo (ex.: "kW").

Insira a unidade desejada, incluindo prefixo (ex.: "MW").

O sistema exibirá um alerta com o novo valor convertido.

Exemplo:

Unidade base: W
Valor: 1500
Unidade atual: W
Unidade desejada: kW

Resultado exibido:
"o novo valor será 1.5 kW"
